**Техническое задание: Мини-сервис заметок**

**Срок на выполнение**: 24 часа с момента получения.

**Цель**  
Создать небольшой веб\-сервис, который позволяет просматривать, создавать, редактировать и удалять заметки. 

Построить RESTful API для сервиса заметок.

**На позицию full-stack/teamlead** требуется дополнить минимальным функциональным UI на React/Vue.

**Для бэкенда** приемлемы любые PHP фреймворки, включая Bitrix, Laravel, Symfony, Yii.

**Требования по API:**  
Реализовать 5 методов: создание, редактирование, удаление, просмотр и вывод всех заметок.  
Каждый метод должен иметь валидацию данных.

**Дополнения** (по желанию, но было бы здорово):  
\- дополнить API тестами на PHPUnit (или альтернатива)  
\- документация swagger (OpenAPI)  
\- использовать docker

Ориентировочное время выполнения задачи 1-2 часа.

**Требования по сдаче (обязательно):**  
\- публичный репозиторий на Github/Gitlab/Bitbucket  
\- подробная инструкция по развёртке приложения (желательно одной командой docker compose up \-d
